Detection fixture: script and iframe host signals

This page exercises script-src and iframe-src host classification. Scripts carry type="text/plain" (never fetched); iframes carry data-src instead of src (no network request). The classifier reads the src/data-src attribute directly from the DOM and maps the hostname to a service.

Signals emitted by this page
Signal type Element / attribute Host extracted Service Category
script-host script[type=text/plain][src] www.googletagmanager.com Google Analytics 4 (via GTM host) analytics
iframe-host iframe[data-src] www.youtube.com YouTube marketing
